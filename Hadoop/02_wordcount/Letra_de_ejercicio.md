
# Ejercicio práctico – Análisis de palabras con Hadoop

## Objetivo

Aplicar el programa **WordCount** de Hadoop para comparar la frecuencia de palabras en distintos textos.

## Instrucciones

### 1. Crear los archivos de texto

1. Crea tres archivos de texto en la carpeta ejemplo_mapreduce: cuento.txt, noticia.txt, song.txt

2. Ejecuta el conteo: hadoopjarhadoop-mapreduce-examples-3.3.6.jar wordcount<archivo>.txt<output>

3. Visualiza los resultados con: cat<output>/part-r-00000

4. Evalúa: ¿Qué texto tiene más repeticiones? ¿Qué palabra aparece más veces?
